# Консольный сканер TCP-портов

Проект реализует консольное Java-приложение для проверки открытых портов на указанном хосте.

## Требования
- Java 11 или новее.

## Сборка и запуск

```bash
cd Java/Semester-5/Final
javac -d out $(find src -name "*.java")
java -cp out com.bachelorlabs.ports.PortScannerApp
```

## Возможности
- `scan <хост> <начальный_порт> <конечный_порт>` — сканирование TCP-портов в диапазоне.
- `help` — вывод подсказки по доступным командам.
- `quit` — завершение работы приложения.

## Примеры
```
> scan example.com 80 443
Открытые порты на хосте example.com:
+ 80 (HTTP)
+ 443 (HTTPS)
```
